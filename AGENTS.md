# Agent instructions

## Design philosophy

This library aims to be a **like-for-like implementation of SCID behaviour**. When making any design or implementation decision — especially around write support, file formats, or API behaviour — the default answer is "do what SCID does". Consult the SCID source at github.com/benini/scid before proposing alternatives.

## Git commits

Do not add `Co-Authored-By` or any other AI attribution trailers to commit messages.
